# cargo-cargo
Happy April Fools' Day!

```
cargo install cargo-cargo
```

And then you can:
```
cargo cargo run
```
Or you can:
```
cargo cargo cargo cargo cargo run
```
Or:
```
cargo cargo cargo cargo cargo cargo cargo cargo cargo cargo run
```
On your project.

Have fun!
